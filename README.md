# U-Shirt! - A t-shirt designer focused on shareability.

<!-- [__Live Demo__](https://aoswalt.github.io/u-shirt) -->

## Setup

Setup: `npm install`

Run: `npm start`

|create design|view own designs|see gallery|view fork history|
